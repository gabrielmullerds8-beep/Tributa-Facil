# Tributa Fácil

MVP de consulta fiscal assistida para operações de agro e mineração.

## O que já faz

- Recebe descrição do produto/operação.
- Sugere NCMs prováveis.
- Permite selecionar origem, destino, regime e tipo de operação.
- Retorna análise preliminar de ICMS, PIS/COFINS, CFEM, IPI, CFOP e CST.

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Aviso

As respostas são sugestões fiscais preliminares. A emissão definitiva deve ser validada por responsável fiscal/contábil.
